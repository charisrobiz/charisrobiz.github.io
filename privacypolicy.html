🦁 Touch Rush - 개인정보 처리방침

최종 수정일: 2024년 9월 4일
시행일: 2024년 9월 4일

1. 개인정보 처리방침 개요

Charisro("회사" 또는 "저희")는 Touch Rush 모바일 애플리케이션("서비스")을 운영합니다. 본 개인정보 처리방침은 저희가 수집하는 정보의 유형, 사용 방법, 그리고 귀하의 개인정보를 보호하는 방법에 대해 설명합니다.

2. 수집하는 정보

2.1 자동으로 수집되는 정보

  • 기기 정보: 기기 모델, 운영체제 버전, 고유 기기 식별자
  • 앱 사용 정보: 게임 진행도, 점수, 플레이 시간
  • 광고 식별자: 맞춤형 광고 제공을 위한 광고 ID

2.2 직접 제공하는 정보

  • 저희 앱은 회원가입이나 개인정보 입력을 요구하지 않습니다
  • 모든 게임 데이터는 기기 내부에 저장됩니다

3. 정보 사용 목적

수집된 정보는 다음 목적으로만 사용됩니다:

  • 서비스 제공: 게임 기능 제공 및 개선
  • 광고 게재: Google AdMob을 통한 맞춤형 광고 표시
  • 분석 및 개선: 앱 성능 분석 및 사용자 경험 개선
  • 기술적 지원: 오류 수정 및 기술적 문제 해결

4. 제3자 서비스

4.1 Google AdMob

저희 앱은 Google AdMob을 사용하여 광고를 표시합니다. Google은 다음 정보를 수집할 수 있습니다:

  • 광고 식별자 (IDFA/GAID)
  • IP 주소
  • 기기 정보
  • 앱 사용 데이터

자세한 내용은 Google 개인정보 처리방침을 참조하세요.

4.2 광고 맞춤설정 제어

개인화된 광고를 원하지 않으시면:

  • iOS: 설정 → 개인정보 보호 → Apple 광고 → 개인화된 광고 제한
  • Android: 설정 → Google → 광고 → 광고 개인 최적화 선택 해제

5. 데이터 보안

저희는 귀하의 정보를 보호하기 위해 다음과 같은 보안 조치를 취합니다:

  • 데이터 암호화 및 안전한 전송
  • 접근 제한 및 권한 관리
  • 정기적인 보안 점검
  • 최소한의 데이터만 수집

6. 아동의 개인정보

Touch Rush는 전체 이용가 게임으로, 13세 미만 아동도 이용할 수 있습니다. 저희는:

  • 아동으로부터 개인정보를 의도적으로 수집하지 않습니다
  • COPPA(아동 온라인 개인정보보호법) 규정을 준수합니다
  • 아동에게 적합하지 않은 광고를 표시하지 않습니다

7. 데이터 보관 기간

  • 게임 데이터: 앱 삭제 시까지
  • 광고 데이터: Google AdMob 정책에 따라 처리
  • 분석 데이터: 최대 2년간 보관 후 자동 삭제

8. 귀하의 권리

귀하는 다음과 같은 권리를 가집니다:

  • 정보 제공 요청: 수집된 개인정보에 대한 정보 요청
  • 정정·삭제 요청: 잘못된 정보의 수정이나 삭제 요청
  • 처리 정지 요청: 개인정보 처리 중단 요청
  • 앱 삭제: 언제든지 앱을 삭제하여 데이터 수집 중단

9. 개인정보 처리방침 변경

본 방침이 변경될 경우:

  • 앱 내 공지사항을 통해 사전 안내
  • 중요한 변경사항은 최소 7일 전 고지
  • 변경된 방침은 공지 후 즉시 적용
